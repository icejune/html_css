宏基因组序列工作流

复制到项目运行

宏基因组学(metagenomics)就是取样---一勺泥土或一杯海水---然后对样品内所有DNA进行测序。这种技术有助于研究人员评估在给定环境发现的生物多样性，但是直到现在它不能有效地对单个物种进行测序，因为没有办法对测得的数据进行解析而区别开每种物种基因组DNA序列。

描述

融合基因或基因嵌合体是改变造成染色体重组结合外显子基因位于同一或不同的染色体。融合基因的产品可能会有一个新的或不同的函数比两个融合的合作伙伴。融合记录经常被发现在不同类型的癌包括乳腺癌、肺癌、前列腺癌,及黑素瘤和淋巴瘤。检测已知的基因融合和识别的小说,可以导致更好的理解致癌作用的触发机制和进展。此外,最近的研究表明,融合基因产品可能代表一个新颖的治疗人类癌症的治疗目标(1)。

有几种融合仪工具,我们已经测试了一组RNA-Seq数据(2)。在这个管道我们使用ChimeraScan软件包,旨在检测融合基因(3)。而找到ChimeraScan展示强劲表现一些假阳性融合检测,该管道只接受paired-end RNA-Seq数据。我们还开发了一种融合检测记录-明星+妄想管道,可以提供快速、充分的性能与循环paired-end读取。更多信息融合发现者,我们测试和用于管道可以在我们的博客很快就会发现。

嵌合体的管道,作为融合的控制记录检测、验证与拉斐尔Calogero教授密切协作的分子生物技术和健康科学,都灵大学(嵌合体作者)。

方法

融合记录检测是使用ChimeraScan软件包包括执行主程序检测融合基因,一个辅助工具,准备参考适当的索引在上游的分析,和工具准备输出HTML表格格式的文件以及格式适用于(3)可视化表示。此外,妄想已添加到这个管道来提供额外的控制检测融合基因(4)。最后,标识的图形表示形式融合基因是由基因组提供的坐标可视化工具Circos(5)。

输入

  • 读(paired-end):该管道接受一对paired-end RNA-Seq FASTQ格式的数据(纯文本或压缩文件)。如果样品中存在多个文件读取,合并FASTQ文件公共管道可用于巩固前对齐。阅读对元数据字段必须设置为1和2。
  • 基因组参考:FASTA文件包含参考基因组。我们建议ucsc.hg19.fasta人类样本。
  • 转录参考:转录参考文件,其中包含所有已知的记录在一个自定义的格式类似于床上。我们建议hg19.ucsc_genes.txt人类样本。
  • 假阳性嵌合体(可选):已知假阳性嵌合体的列表。我们建议hg19_bodymap_false_positive_chimeras.txt人类样本。

工具和建议的参数设置

  • ChimeraScan v0.4.5——软件包旨在检测基因融合paired-end RNA-Seq数据集(3)。ChimeraScan使用领结对齐paired-end读结合基因组,转录参考,旨在发现不和谐的读取,预测最优融合断点位置和检测嵌合体。软件包包括一个索引程序,ChimeraScan指数创建组合索引从基因组参考序列(FASTA格式)和自定义转录参考格式(UCSC的GenePred格式)。注意:当使用GTF转录文件,ChimeraScan GTF genePred工具可以被添加到这个管道将这个文件的格式可以接受由ChimeraScan指数。ChimeraScan HTML表,允许输出生成的HTML页面的链接嵌合基因的详细描述。此外,Circos suitab /名称生成器生成输出文件的链接
    • 参数设置为默认值基于ChimeraScan作者建议的最佳实践。
